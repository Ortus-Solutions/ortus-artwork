# ORM EXTENSION

**ORM EXTENSION** is an advanced extension for enhancing ORM capabilities within the Ortus Solutions ecosystem, providing additional utilities, integrations, and productivity features for working with database entities and object-relational mapping workflows.

## 📝 Notes

- Folders included: `SVG`, `PNG`, `JPG`
- File naming convention:  
  `ormextension-[logo|icon]-[variant]-[tone]-[size].[format]`
- Sizes available: **L**, **M**, **S**
- Use **SVG** whenever possible for scalability and best quality.
- Use **PNG** for digital applications.
- Use **JPG** only when required for compatibility (except Mono Light, which does not include JPG versions).
