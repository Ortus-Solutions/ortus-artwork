## 📝 Notes

- Folder structure: `SVG`, `PNG`, `JPG`
- Naming convention: `cfcasts-[logo|icon]-[variant]-[tone]-[size].[format]`
- Sizes: **L, M, S**
- JPG files are only included for **light variants**
