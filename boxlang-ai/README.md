## 📝 Notes

- Use **Full Color** logos whenever possible.
- Use **Monochrome** versions only when color usage is restricted.
- Do not alter proportions, colors, or typography.
- File naming convention follows:  
  `bxai-[logo]-[variant]-[tone]-[size].[format]`  
  `boxlang-icon-[variant]-[tone]-[size].[format]`
